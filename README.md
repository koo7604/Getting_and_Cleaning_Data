# How to reproduce the result?
1. Folk and clone this repository. (https://github.com/koo7604/Getting_and_Cleaning_Data)
2. Open the file "run_analysis.R".
3. Run all the codes there. The data comes from "./Getting_and_Cleaning_Data", "./Getting_and_Cleaning_Data/train", and "./Getting_and_Cleaning_Data/test".
4. The output file is "tinydataset.txt".
